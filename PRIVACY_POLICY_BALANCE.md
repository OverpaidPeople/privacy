## 2. 처리하는 정보

### 2.1 기기에만 저장되는 포트폴리오 정보

사용자가 직접 입력한 다음 정보는 원칙적으로 사용자의 기기 내부 데이터베이스에 저장됩니다.

- 보유 종목과 현금 자산
- 보유 수량
- 평균 매입 단가
- 목표 비중
- 거래 내역
- 배당 내역
- 최근 검색 및 관심 종목
- 앱 설정과 백테스트 임시 결과

이 정보는 밸런스 운영 서버의 사용자 데이터베이스에 저장되지 않습니다.

사용자가 Android 공유 기능이나 백업 기능을 직접 실행한 경우에는 사용자가 선택한 대상 또는 저장 위치로 정보가 전달될 수 있습니다. 이러한 전송은 사용자의 명시적인 동작에 따라 이루어집니다.

### 2.2 Firebase 익명 인증 정보

앱은 외부인의 무분별한 API 호출을 방지하고 정상적인 앱 요청을 식별하기 위해 Firebase Authentication의 익명 인증을 사용합니다.

이 과정에서 다음 정보가 처리될 수 있습니다.

- Firebase 익명 사용자 식별자(UID)
- IP 주소
- 앱 식별 정보
- 기기 및 운영체제 관련 정보
- Firebase 인증 토큰과 인증 요청 정보

앱은 익명 인증 과정에서 사용자의 이름, 이메일 주소, 전화번호 또는 비밀번호를 직접 입력받지 않습니다.

### 2.3 앱 사용 및 분석 정보

앱은 서비스 품질 분석을 위해 Firebase Analytics를 사용합니다. 다음 정보가 자동으로 수집될 수 있습니다.

- 앱 실행 및 화면 이용 정보
- 기능 사용 및 앱 상호작용 정보
- 앱 버전
- 기기 모델과 운영체제 버전
- 언어와 국가 등의 일반적인 기기 정보
- 앱 인스턴스 또는 기기 관련 식별자

수집되는 정보는 앱 이용 현황 분석과 서비스 개선에 사용됩니다.

### 2.4 오류 및 진단 정보

앱은 오류 확인과 안정성 개선을 위해 Firebase Crashlytics를 사용합니다. 다음 정보가 자동으로 수집될 수 있습니다.

- 비정상 종료 및 오류 로그
- 오류가 발생한 코드 위치와 스택 정보
- 앱 상태와 앱 버전
- 기기 모델과 운영체제 버전
- 오류 발생 시각
- 앱 인스턴스 또는 설치 관련 식별자

### 2.5 광고 관련 정보

앱은 Google AdMob을 통해 광고를 제공할 수 있습니다. Google Mobile Ads SDK는 광고 제공, 광고 성과 측정, 분석 및 부정 사용 방지를 위해 다음 정보를 자동으로 수집하거나 공유할 수 있습니다.

- IP 주소 및 이를 통해 추정되는 대략적인 위치
- 광고 ID와 앱 세트 ID 등 기기 또는 계정 관련 식별자
- 앱 실행, 광고 조회, 탭 등의 앱 상호작용 정보
- 앱과 광고 SDK의 성능 및 진단 정보
- 기기 및 운영체제 관련 정보

법령상 동의가 필요한 지역에서는 Google User Messaging Platform(UMP)을 통해 광고 개인정보 선택 사항과 동의를 요청할 수 있습니다.

### 2.6 주식 정보 API 이용 정보

앱은 종목 검색, 시세, 차트, 환율, 뉴스와 기업정보를 제공하기 위해 밸런스 API 서버와 통신합니다.

이 과정에서 다음 정보가 서버에 전달되거나 일시적으로 처리될 수 있습니다.

- Firebase 익명 사용자 식별자와 인증 토큰
- IP 주소
- 요청 시각과 요청 경로
- 사용자가 조회한 검색어, 종목 코드 또는 티커
- 앱의 네트워크 요청 및 오류 정보

밸런스 API 서버는 사용자의 보유 수량, 평균 매입 단가, 목표 비중, 거래 내역 또는 배당 내역을 저장하지 않습니다.

해당 정보는 서비스 제공, 보안, 분석, 오류 진단 및 광고 제공에 필요한 기간 동안 각 서비스 제공자의 정책과 관련 법령에 따라 처리됩니다.

앱은 금융상품의 매매를 중개하거나 특정 투자를 보장하지 않습니다. 실제 투자 판단과 거래 결정에 대한 책임은 사용자에게 있습니다.
